테트리스 게임의 기본 구조를 구현하였습니다.

### 생성/수정한 파일 목록
- `src/index.html`: 게임 화면 레이아웃
- `src/style.css`: 게임 보드 및 UI 스타일
- `src/script.js`: 기본 블록 렌더링 및 키보드 조작 로직

### 구현한 기능
- HTML5 Canvas를 이용한 20x10 게임 보드 렌더링
- 키보드 방향키를 이용한 블록 이동(좌, 우, 하)
- 기본 블록(T자 모양) 렌더링 테스트

### 테스트 결과
- `src/index.html`을 실행하면 캔버스에 블록이 표시되며, 방향키 입력을 통해 블록의 위치가 변경되는 것을 확인하였습니다.

이후 상세 로직(충돌 검사, 라인 삭제, 게임 루프 등)을 추가하여 계획하신 모든 기능을 완성할 수 있습니다. 추가적인 기능 구현이 필요하시면 말씀해 주세요.
