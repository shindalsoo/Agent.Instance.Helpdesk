# 프로젝트 개요
- **목표**: HTML/CSS/JavaScript만으로 웹에서 실행 가능한 최소 기능의 테트리스(Tetris) 게임을 구현한다.
- **핵심 기능**
  - 블록이 아래로 낙하하고 회전, 이동(좌우)
  - 라인 완성 시 삭제 및 점수 증가
  - 게임 오버 처리
  - 스코어와 레벨 표시
  - 간단한 UI(시작/재시작 버튼)

# 기술 스택
- **프론트엔드**  
  - `HTML5` – 구조, 캔버스 혹은 그리드 기반 렌더링  
  - `CSS3` – 스타일링 및 애니메이션 (필요 시)  
  - `Vanilla JavaScript (ES6+)` – 게임 로직, 이벤트 처리
- **툴**  
  - 웹 브라우저(크롬/파이어폭스 등)만으로 실행 가능

# 파일 구조
```
src/
├─ index.html          # 메인 페이지
├─ style.css           # 스타일시트
└─ script.js           # 게임 로직
```

> **주의**: 모든 소스는 `src/` 폴더 아래에 위치해야 합니다.

# 기능 명세

## 1. UI 레이아웃 (`index.html`, `style.css`)
| 요소 | 설명 |
|------|------|
| `<canvas>` 혹은 그리드 컨테이너 | 게임 보드를 표시 (20x10 셀) |
| 점수 영역 | 현재 스코어, 레벨 표시 |
| 시작/재시작 버튼 | 게임 초기화 및 재시작 기능 |

## 2. 블록 정의 (`script.js`)
- **형상**: I, O, T, S, Z, J, L
- **색상**: 각 블록마다 고유 색상 배열
- **회전**: `rotate()` 메서드로 시계 방향 회전 (4개의 모양)

## 3. 보드 관리 (`script.js`)
- 2차원 배열(`board[20][10]`)으로 현재 상태 저장
- 충돌 검사(`isColliding()`) – 블록이 벽, 다른 블록에 닿았는지 판단
- 라인 완성 검사(`clearLines()`) – 가득 찬 행 삭제 후 위쪽 행들 내려오게 함

## 4. 게임 루프 (`script.js`)
- `requestAnimationFrame` 또는 타이머를 이용해 일정 시간마다 블록을 한 칸씩 아래로 이동
- 사용자의 키보드 입력(`keydown`)으로 좌우 이동, 회전, 즉시 낙하 처리
- 충돌 시 현재 블록을 보드에 고정하고 새로운 블록 생성

## 5. 점수 & 레벨 (`script.js`)
- 라인 한 줄 삭제: 40점  
- 두 줄: 100점  
- 세 줄: 300점  
- 네 줄(블록): 1200점
- 레벨은 `score / 1000` (정수)로 계산하며, 레벨이 올라갈 때마다 낙하 속도 증가

## 6. 게임 오버 처리 (`script.js`)
- 새 블록 생성 시 충돌 발생 → 게임 종료
- 종료 화면에 “Game Over” 표시 및 재시작 버튼 활성화

## 7. 저장 & 불러오기 (선택, 미구현)
- 로컬 스토리지 활용해 최고 점수 저장(옵션)

# 구현 우선순위
1. **프로젝트 구조 설정** – 파일 생성 및 기본 HTML/CSS skeleton 작성  
2. **보드 렌더링** – 그리드 기반 보드 표시 (CSS Grid 혹은 Canvas)  
3. **블록 정의 & 회전 로직** – 데이터 모델과 회전 함수 구현  
4. **게임 루프 & 충돌 검사** – 블록 낙하 및 이동 처리  
5. **라인 삭제 & 점수 계산** – 라인 완성 시 보드 업데이트, 스코어 표시  
6. **UI 추가** – 시작/재시작 버튼, 레벨/점수 UI  
7. **게임 오버 처리** – 종료 조건과 화면 표시  
8. **미세 조정** – 속도 튜닝, 애니메이션, 코드 정리

> **Tip**: 각 단계별로 작은 테스트를 진행하며 기능이 정상 동작하는지 확인하세요.
