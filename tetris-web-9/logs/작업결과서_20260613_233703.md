웹 테트리스 게임 구현을 완료하였습니다.

### 생성/수정한 파일 목록
- `src/index.html`: 게임 구조 및 UI 레이아웃
- `src/style.css`: 게임 보드 및 UI 스타일링
- `src/script.js`: 게임 로직(블록 생성, 이동, 회전, 충돌, 라인 삭제, 게임 루프 등)

### 구현한 기능
- **블록 시스템**: I, O, T, S, Z, J, L 7종의 블록 구현
- **게임 로직**: 키보드 입력(좌우 이동, 회전) 및 자동 낙하 루프
- **보드 관리**: 20x12 그리드 기반의 충돌 검사 및 라인 삭제 로직
- **점수 및 레벨**: 라인 제거 시 점수 획득 및 점수에 따른 레벨 표시
- **UI**: 시작 버튼을 통해 게임을 제어하고 점수/레벨을 실시간으로 업데이트

### 테스트 결과
- `src/index.html` 파일을 브라우저에서 열어 정상적으로 게임이 실행되는 것을 확인하였습니다.
- 키보드 방향키(좌/우/아래)와 회전 키(Q/W)를 통해 블록을 조작할 수 있습니다.
- 블록이 벽과 바닥에 닿았을 때 충돌 처리가 정확히 작동합니다.
- 한 줄이 완성되면 보드에서 제거되고 위쪽 블록들이 내려오며 점수가 계산됩니다.
- 게임 오버 시 보드가 초기화됩니다.

이제 모든 작업이 완료되었습니다.
