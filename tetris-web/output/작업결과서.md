테트리스 게임 개발을 완료하였습니다.

### 생성/수정한 파일 목록
- `src/index.html`: 게임 메인 구조
- `src/css/style.css`: 게임 스타일링
- `src/js/constants.js`: 게임 상수(블록 모양, 색상 등) 정의
- `src/js/utils.js`: 유틸리티 함수
- `src/js/game.js`: 핵심 게임 로직(엔진, 루프, 충돌, 라인 클리어)
- `src/js/input.js`: 키보드 입력 핸들러
- `src/js/ui.js`: UI 업데이트 함수

### 구현한 기능 설명
- **핵심 게임 엔진**: Canvas를 활용하여 10x20 그리드 기반의 게임 루프를 구현하였습니다.
- **블록 로직**: 7가지 테트로미노 생성, 회전(벽 차기 로직 포함), 이동, 중력 하강 기능을 구현했습니다.
- **충돌 감지**: 벽과 바닥, 이미 배치된 블록과의 충돌을 처리합니다.
- **게임 규칙**: 한 줄이 찼을 때 라인을 제거하고 점수를 계산하는 시스템을 구현하였습니다.
- **조작**: 방향키(좌/우/아래)와 회전 키(Q, W)를 통해 조작 가능합니다.

### 테스트 결과
- 브라우저에서 `src/index.html`을 열어 정상적으로 게임이 시작되는 것을 확인했습니다.
- 블록이 아래로 이동하고, 좌우 이동 및 회전이 의도대로 동작합니다.
- 라인이 찼을 때 제거되고 점수가 올라가는 것을 확인했습니다.
