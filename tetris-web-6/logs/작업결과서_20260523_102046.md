이 프로젝트는 웹 기반 테트리스 게임을 개발하는 것을 목표로 합니다. 현재까지의 진행 상황은 다음과 같습니다.

### 생성/수정한 파일 목록
- `index.html`: 프로젝트 엔트리 포인트 및 구조 설정
- `src/css/style.css`: 게임 레이아웃 및 스타일링
- `src/js/constants.js`: 게임 상수(그리드 크기, 색상 등) 정의
- `src/js/tetrominoes.js`: 테트로미노 정의 및 회전 로직 기초
- `src/js/engine.js`: 게임 엔진 클래스 및 렌더링 로직
- `src/js/ui.js`: UI 업데이트 함수
- `src/js/main.js`: 게임 초기화 및 시작 로직

### 구현한 기능 설명
- 프로젝트 폴더 구조를 설정하고 HTML/CSS 기반의 UI 스캐폴딩을 완성했습니다.
- 테트리스 게임의 핵심인 10x20 그리드 시스템과 테트로미노 데이터를 정의했습니다.
- HTML5 Canvas를 사용하여 화면에 그리드와 테트로미노를 그리는 기초 엔진을 구현했습니다.

### 테스트 결과
- 브라우저에서 `index.html`을 열었을 때, 테트리스 게임의 기본 틀(캔버스, 점수판)이 정상적으로 렌더링되며, 첫 번째 테트로미노가 화면에 나타나는 것을 확인했습니다.
