테트리스 게임 개발 프로젝트가 완료되었습니다. 최종 결과물에 대한 요약 보고입니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 구조 및 레이아웃
- `src/css/style.css`: 게임 스타일링 및 UI 레이아웃
- `src/js/constants.js`: 게임 상수(그리드, 색상) 정의
- `src/js/tetrominoes.js`: 테트로미노 데이터 및 회전 로직
- `src/js/engine.js`: 게임 핵심 엔진(중력, 충돌, 라인 삭제, 점수 시스템)
- `src/js/ui.js`: UI 업데이트(점수판)
- `src/js/main.js`: 게임 초기화, 사용자 입력 처리 및 게임 루프

### 구현한 기능
1.  **핵심 게임 엔진**: 10x20 그리드 시스템, `requestAnimationFrame`을 이용한 부드러운 게임 루프, 자동 중력 및 블록 고정 로직 구현.
2.  **조작 및 물리**: 화살표 키를 사용한 이동 및 회전(벽 킥 로직 포함), 충돌 감지 시스템.
3.  **게임 규칙**: 가득 찬 라인 삭제 로직, 점수 계산 시스템(1~4라인), 실시간 점수 업데이트.
4.  **UI/UX**: 다음 블록 미리보기(Next Piece), 실시간 점수판, 게임 오버 시 자동 초기화.

### 테스트 결과
- 브라우저에서 `index.html`을 실행하여 전체 게임 루프가 정상적으로 작동함을 확인했습니다.
- 블록 이동, 회전, 충돌, 라인 삭제 및 점수 획득이 기획된 대로 작동합니다.
- 다음 블록 미리보기 기능이 정상적으로 구현되었습니다.

프로젝트가 성공적으로 마무리되었습니다. 추가적인 기능(난이도 상승 시스템, 하드 드롭 등)은 향후 업데이트를 통해 보완할 수 있습니다.
