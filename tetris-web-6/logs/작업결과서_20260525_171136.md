테트리스 게임의 충돌 버그를 수정하였습니다.

### 수정한 파일
- `src/js/engine.js`

### 구현한 기능 설명
- `collide` 메서드 내의 충돌 감지 로직을 수정하였습니다.
- 기존 로직에서는 블록의 일부가 게임 보드 위쪽(y < 0)으로 나갈 경우, 이를 무조건 충돌로 간주하여 게임이 비정상적으로 종료되거나 블록이 고정되는 문제가 있었습니다.
- 수정을 통해 블록이 보드 영역 안으로 들어오기 전(y < 0)에는 충돌 감지를 건너뛰고, 보드 영역 내에 있을 때만 충돌을 체크하도록 개선하였습니다.

### 테스트 결과
- 블록이 생성될 때 보드 상단 영역에서도 자유롭게 이동할 수 있음을 확인하였습니다.
- 벽이나 기존 블록과의 충돌은 정상적으로 감지되어 게임 로직이 올바르게 작동합니다.
