웹 기반 테트리스 게임 개발을 완료했습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 구조 및 레이아웃
- `src/css/style.css`: 게임 스타일링
- `src/js/constants.js`: 게임 상수 정의
- `src/js/tetrominoes.js`: 테트로미노 데이터 및 회전 로직
- `src/js/engine.js`: 게임 로직(중력, 충돌, 라인 삭제, 점수)
- `src/js/ui.js`: UI 업데이트 함수
- `src/js/main.js`: 게임 초기화 및 이벤트 리스너

### 구현한 기능
- **게임 엔진**: 10x20 그리드 시스템, 자동 중력, 충돌 감지 및 블록 고정.
- **조작**: 화살표 키를 이용한 이동 및 회전.
- **게임 규칙**: 라인 삭제 시 점수 획득(1줄: 100점, 2줄: 300점, 3줄: 500점, 4줄: 800점).
- **UI/UX**: 다음 블록 미리보기(Next Piece), 실시간 점수판.

### 테스트 결과
- 브라우저에서 `index.html`을 실행하여 전체 게임 루프가 원활하게 작동함을 확인했습니다.
- 블록이 하단이나 다른 블록과 충돌 시 정확히 멈추고, 가득 찬 라인이 삭제되며 점수가 업데이트되는 것을 확인했습니다.
- 다음 블록 미리보기 기능이 정상적으로 작동합니다.
