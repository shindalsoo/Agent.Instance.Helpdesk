### 수정된 파일
- `src/js/engine.js`: `Game` 클래스에 `gameOver` 상태를 추가하고, `draw` 메서드에서 게임 오버 메시지를 렌더링하도록 수정했습니다. 또한 게임 재시작을 위한 `reset` 메서드를 추가했습니다.
- `src/js/main.js`: `playerReset` 함수에서 게임 오버 시 즉시 초기화하는 대신 `game.gameOver` 상태를 `true`로 설정하도록 변경했습니다. `keydown` 이벤트 핸들러에 게임 오버 상태일 때 아무 키나 누르면 `game.reset()`을 호출하여 게임이 재개되도록 수정했습니다.

### 구현한 기능
- 게임 오버 시 화면에 'GAME OVER' 메시지와 'Press any key to restart' 안내 문구를 표시.
- 게임 오버 상태에서 사용자가 키를 입력하면 게임이 처음부터 다시 시작되도록 기능 구현.

### 테스트 결과
- 게임 오버 조건(테트리스 블록이 상단에 도달) 발생 시 게임이 즉시 멈추고 안내 메시지가 표시됨을 확인했습니다.
- 이후 키보드 입력을 통해 게임이 정상적으로 초기화되고 재개되는 것을 확인했습니다.
