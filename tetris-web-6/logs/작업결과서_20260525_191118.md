테트리스 게임에 '하드 드롭(Hard Drop)' 기능을 성공적으로 구현하였습니다.

### 수정한 파일 목록
- `src/js/engine.js`: `playerHardDrop()` 메서드를 추가하여 블록이 바닥까지 즉시 떨어지고 고정되도록 구현했습니다.
- `src/js/main.js`: `keydown` 이벤트 리스너에 엔터(Enter, keyCode 13) 키 입력을 추가하여 `game.playerHardDrop()`을 호출하도록 연결했습니다.

### 구현한 기능 설명
- **하드 드롭(Hard Drop)**: 사용자가 엔터(Enter) 키를 누르면 현재 조작 중인 블록이 충돌 지점까지 즉시 내려가고, 보드에 고정된 후 다음 블록이 생성되는 기능을 구현했습니다. `engine.js`에서 `while` 루프를 사용하여 충돌이 발생하기 전까지 블록의 y 좌표를 증가시키고, 충돌 시 `merge`와 `playerReset`을 수행하도록 작성했습니다.

### 테스트 결과
- 게임 플레이 중 엔터 키를 누르면 블록이 즉시 바닥으로 이동하며 고정되는 것을 확인했습니다.
- 기존의 좌/우 이동, 회전, 소프트 드롭 기능이 정상적으로 작동함을 확인했습니다.
