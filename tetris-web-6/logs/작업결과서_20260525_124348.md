테트리스 게임의 우측 경계 충돌 버그를 수정하였습니다.

### 수정한 파일
- `src/js/engine.js`

### 구현한 기능 설명
- `Game.prototype.collide` 메서드 내의 충돌 감지 로직을 보강하였습니다.
- 기존 코드에서는 `this.grid[y + o.y][x + o.x]`의 존재 여부만 체크하고 있어, 그리드 범위를 벗어나는 경우에 대한 명시적인 경계 체크가 부족했습니다.
- `x + o.x < 0 || x + o.x >= COLS || y + o.y >= ROWS` 조건을 추가하여 도형이 게임 보드의 좌우 및 하단 경계를 벗어나지 않도록 수정했습니다.

### 테스트 결과
- 로직 수정 후, 게임의 모든 열(0번부터 9번까지)에서 도형이 정상적으로 이동하고, 우측 끝에서도 경계를 넘지 않고 충돌 감지가 올바르게 작동함을 확인하였습니다.
