테트리스 게임의 충돌 감지 로직을 수정하여 버그를 해결했습니다.

### 수정/생성된 파일
- `src/game.js`

### 구현한 기능 설명
- **충돌 감지 로직 개선**:
    - `validMove` 함수에서 `newY < 0`인 경우를 허용하여 테트로미노가 게임판 상단에서 생성될 때 충돌로 오인하지 않도록 수정했습니다.
    - 테트로미노가 게임판 내부에 있을 때만 `grid`와의 충돌을 검사하도록 `if (newY >= 0 && grid[newY][newX])` 조건을 추가했습니다.
- **게임 로직 안정화**:
    - 테트로미노 생성 시 초기 위치를 조정했습니다.
    - `lock` 함수에서 게임판 영역 밖(상단)인 경우를 안전하게 처리하도록 변경했습니다.
    - `rotatePiece`에서 Wall Kick 로직을 더 견고하게 수정했습니다.
    - `init` 함수에서 `setInterval` 시간을 조절하여 레벨 증가에 따른 속도 증가를 보장했습니다.

### 테스트 결과
- 코드 수정 후, 테트로미노가 상단에서 생성될 때 즉시 충돌로 처리되던 문제가 해결되었습니다.
- 벽 근처에서의 회전 및 이동이 정상적으로 작동함을 확인했습니다.
- 라인 제거 및 점수 계산 로직이 정상적으로 유지됩니다.
