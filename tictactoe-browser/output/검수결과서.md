## 파일 구조

```
tic-tac-toe/
└── src/
    ├── index.html
    ├── styles.css
    └── script.js
```

- **index.html** – 페이지 레이아웃 및 스크립트/스타일 링크
- **styles.css** – 그리드, 셀 스타일, 승리 시 하이라이트
- **script.js** – 게임 로직(상태, 턴, 감지, 리셋)

---

## 사용 방법

1. `src/index.html` 파일을 브라우저에서 엽니다.
2. X/O 두 플레이어가 번갈아 가며 셀을 클릭합니다.
3. 승리 시 녹색 하이라이트와 메시지가 표시됩니다.
4. 무승부 시 “Draw!” 메시지 표시.
5. `Reset Game` 버튼으로 새 게임 시작.
