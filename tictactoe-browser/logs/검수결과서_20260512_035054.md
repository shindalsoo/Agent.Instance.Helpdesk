## ✅ Created Files

| Path | Description |
|------|-------------|
| `src/index.html` | Main HTML page with board, status bar and reset button |
| `src/styles.css` | CSS for layout, cell styling, win highlight |
| `src/script.js` | Vanilla JS implementing game logic, turn handling, win/draw detection & reset |

## 🎮 How It Works

1. **Board Generation** – 9 `<button>` elements are created dynamically and placed in a 3×3 grid.
2. **Turn Handling** – `isXTurn` toggles after each valid move; status bar shows the current player.
3. **Move Validation** – Clicking an empty cell records the mark (`X` or `O`). After a move, the game checks for a win or draw.
4. **Win Detection** – All 8 winning line combinations are checked; if matched, cells get the `.win` class (green background) and the status bar announces the winner.
5. **Draw Detection** – When all cells are filled with no winner, “Draw!” is shown.
6. **Reset** – Clears board, removes win highlights, resets state, and sets turn back to Player X.

Open `src/index.html` in any modern browser to play. No build steps or dependencies required.
