## 생성·수정한 파일 목록
| 경로 | 설명 |
|------|------|
| `src/index.html` | 3×3 보드, 상태 표시줄, 리셋 버튼이 포함된 메인 페이지 |
| `src/styles.css` | 그리드 레이아웃, 셀 스타일링 및 승리 시 하이라이트를 담당하는 CSS |
| `src/script.js` | 게임 로직(상태 관리, 턴 전환), 승리/무승부 감지 및 리셋 기능을 구현한 순수 자바스크립트 |

---

## 구현한 기능
1. **보드 자동 생성** – 스크립트에서 9개의 `<button>`을 동적으로 만들어 그리드에 배치합니다.
2. **턴 전환** – `isXTurn` 플래그를 토글하며, 현재 플레이어를 상태 표시줄에 보여줍니다.
3. **이동 유효성 검사** – 이미 채워진 셀이나 게임 종료 후 클릭을 무시합니다.
4. **승리 감지** – 8개의 승리 라인(가로·세로·대각선)을 검사하고, 일치하면 해당 셀에 `win` 클래스를 부여해 색상을 변경하며 승자 메시지를 표시합니다.
5. **무승부 감지** – 모든 셀이 채워졌는데 승자가 없으면 “Draw!”를 출력합니다.
6. **리셋 기능** – 보드와 상태를 초기화하고, 승리 하이라이트를 제거해 새 게임을 시작할 수 있게 합니다.

---

## 테스트 결과
- `src/index.html` 파일을 브라우저에서 열면 바로 게임이 실행됩니다.
- X/O 두 플레이어가 번갈아 가며 셀을 클릭하면 마크가 표시되고, 승리/무승부가 정확히 감지됩니다.
- 승리 시 해당 세 개의 셀이 녹색 배경으로 하이라이트되고, 상태 표시줄에 승자 메시지가 나타납니다.
- 리셋 버튼을 누르면 보드가 비워지고 모든 상태가 초기화됩니다.

모든 기능이 정상적으로 동작하며, 추가적인 의존성 없이 순수 HTML/CSS/JS 로 구성되었습니다.
