### 생성/수정한 파일 목록
- `src/constants.js`: 게임 설정값 정의
- `src/index.html`: 게임 화면 UI 구조
- `src/style.css`: 게임 스타일링 및 그리드 레이아웃
- `src/script.js`: 게임 초기화 및 타일 생성 로직

### 구현한 기능 설명
- **4x4 그리드 구조**: HTML/CSS Grid를 사용하여 2048 게임판을 구현했습니다.
- **게임 로직**: 초기 타일 2개 생성(2 또는 4) 및 타일 생성 알고리즘이 적용되었습니다.
- **게임 제어**: 'New Game' 버튼을 통해 게임을 재시작할 수 있습니다.
- **상태 관리**: JavaScript 모듈을 사용하여 상수를 관리하고 로직을 분리하였습니다.
