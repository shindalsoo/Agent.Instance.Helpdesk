사용자의 요청에 따라 `src/script.js` 파일을 수정하여 화살표 키 입력 시 브라우저 기본 스크롤 동작을 방지하고 게임 로직이 정상적으로 호출되도록 개선했습니다.

### 수정한 파일
- `src/script.js`: `keydown` 이벤트 리스너 내부에 `e.preventDefault()`를 추가하여 방향키 입력 시 페이지가 스크롤되는 것을 방지하고 게임 로직(`move`)이 정상적으로 동작하도록 했습니다.

### 구현한 기능 설명
- **키 입력 처리 개선**: 방향키(ArrowUp, ArrowDown, ArrowLeft, ArrowRight) 입력 시 `e.preventDefault()`를 호출하여 브라우저의 기본 동작을 막고, 게임 내 이동 로직(`move`)이 정확히 실행되도록 했습니다.

### 테스트 결과
- 브라우저 환경에서 방향키를 입력했을 때 페이지 스크롤 없이 게임판의 타일들이 정상적으로 이동하고 병합되는 것을 확인했습니다.
