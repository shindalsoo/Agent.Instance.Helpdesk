2048 퍼즐 게임 개발을 완료하였습니다.

### 생성/수정한 파일 목록
- `src/constants.js`: 게임 설정 값 정의
- `src/index.html`: 게임 기본 구조
- `src/style.css`: 게임 스타일링 및 타일별 색상/애니메이션
- `src/script.js`: 핵심 게임 로직(그리드, 이동, 병합, 키보드 이벤트)

### 구현한 기능 설명
- **핵심 게임 로직**: 4x4 그리드 기반의 타일 병합 알고리즘 구현.
- **방향 제어**: 키보드 방향키(상/하/좌/우)를 통한 타일 이동 및 병합.
- **점수 시스템**: 타일이 합쳐질 때마다 점수를 누적하고 화면에 표시.
- **직관적 UI**: CSS Grid를 사용하여 타일 정렬, `data-value` 속성을 활용한 숫자별 색상 스타일링.

### 테스트 결과
- 브라우저에서 `src/index.html`을 열면 4x4 그리드가 표시됩니다.
- 초기 2개의 타일이 무작위 위치에 생성됩니다.
- 방향키 입력 시 타일이 정상적으로 슬라이딩되고, 같은 숫자가 만나면 합쳐집니다.
- 합쳐진 후에는 새로운 타일(2 또는 4)이 무작위 빈칸에 생성됩니다.
- 기본적인 게임 플레이 로직이 정상적으로 작동함을 확인했습니다.
