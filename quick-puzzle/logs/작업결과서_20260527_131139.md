2048 퍼즐 게임 개발을 완료했습니다.

### 생성/수정한 파일 목록
- `src/index.html`: 게임의 구조 및 레이아웃 정의
- `src/style.css`: 게임 UI 스타일링 및 타일 색상 표현
- `src/script.js`: 게임 핵심 로직(이동, 병합, 점수 관리) 구현

### 구현한 기능 설명
1.  **게임 엔진**: 4x4 격자판을 기반으로 타일 생성, 이동, 병합 로직을 구현했습니다.
2.  **점수 시스템**: 타일이 병합될 때마다 점수가 누적되며, `localStorage`를 사용하여 최고 점수를 저장합니다.
3.  **UI/UX**: HTML/CSS를 사용하여 깔끔하고 직관적인 게임 화면을 구성했습니다.
4.  **입력 처리**: 키보드 방향키를 통해 게임을 조작할 수 있습니다.
5.  **재시작 기능**: 게임 완료 후 'New Game' 버튼을 통해 즉시 초기화가 가능합니다.

### 테스트 결과
- 브라우저에서 `src/index.html`을 열어 정상적으로 게임이 렌더링되는 것을 확인했습니다.
- 키보드 방향키 입력에 따라 타일이 이동하고, 동일한 숫자가 만났을 때 병합되는 로직이 정확히 작동합니다.
- 점수판과 최고 점수 저장 기능이 정상적으로 동작합니다.

별도의 복잡한 설치 없이 브라우저에서 바로 실행 가능한 형태로 완성되었습니다.
