생성/수정한 파일 목록:
- `src/script.js`: 게임 상태(History Stack)를 관리하는 `history` 배열을 추가하고, 이동할 때마다 이전 상태를 저장하는 `saveState` 함수를 구현했습니다. 또한, 'Undo' 버튼 클릭 시 직전 상태로 복구하는 `undo` 함수를 추가했습니다.
- `src/index.html`: UI에 'Undo' 버튼을 추가하고, 기존의 'New Game' 버튼과 함께 레이아웃을 정리했습니다.

구현한 기능 설명:
- **실행 취소(Undo) 기능**: 유저가 방향키를 입력하여 타일이 이동/병합되기 직전의 `grid` 상태와 `score`를 `history` 스택에 저장합니다. 'Undo' 버튼을 누르면 스택에서 최근 상태를 가져와 게임 보드와 점수를 복구합니다. (최근 20회까지의 이동 기록을 유지합니다.)

테스트 결과:
- 게임 플레이 중 방향키를 사용하여 타일을 이동시킨 뒤, 'Undo' 버튼을 클릭하면 이전 상태로 정상적으로 되돌아감을 확인했습니다.
- 점수 또한 이동 전의 점수로 복구됩니다.
