### 생성/수정한 파일 목록
- `src/index.html`: 게임 메인 구조
- `src/css/style.css`: 게임 스타일 및 레이아웃
- `src/js/engine.js`: 2048 게임 엔진 로직
- `src/js/ui.js`: DOM UI 관리
- `src/js/main.js`: 게임 초기화 및 입력 처리

### 구현한 기능 설명
- **4x4 그리드 시스템**: 2차원 배열을 통한 게임 상태 관리.
- **기본 엔진**: 타일 생성(2 또는 4), 오른쪽 방향 이동 및 병합 로직 구현.
- **UI 렌더링**: 현재 그리드 상태를 DOM에 반영하고 점수를 표시.
- **입력 처리**: 방향키(오른쪽)를 통한 게임 조작.

### 테스트 결과
- 브라우저 환경에서 `index.html`을 열어 게임이 정상적으로 로드됨을 확인했습니다.
- 오른쪽 방향키 입력 시 타일이 이동하고 병합되는 것을 확인했습니다.
- CSS 스타일을 통해 타일의 기본 색상 및 레이아웃이 적용되었습니다.

*참고: 현재는 오른쪽 방향 이동만 구현되어 있습니다. 나머지 방향 로직은 `engine.js`의 `move` 함수를 확장하여 동일한 패턴으로 구현 가능합니다.*
