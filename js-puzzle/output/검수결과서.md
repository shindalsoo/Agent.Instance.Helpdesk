### 구현한 기능 설명
- **게임 상태 관리**: 더 이상 이동할 수 없는 경우(패배)와 2048 타일을 만든 경우(승리)를 감지합니다.
- **게임 종료/승리 화면**: 게임 종료 시 오버레이를 통해 결과를 알리고 '다시 시작' 버튼을 제공합니다.
- **재시작 기능**: 게임 상태를 초기화하고 화면을 갱신합니다.

### 테스트 결과
- 게임이 더 이상 진행 불가능한 상태가 되면 '게임 오버!' 메시지가 표시됨을 확인했습니다.
- '다시 시작' 버튼을 클릭하면 게임이 정상적으로 초기화되고 다시 시작함을 확인했습니다.
