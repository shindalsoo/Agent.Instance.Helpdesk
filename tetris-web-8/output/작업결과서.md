요청하신 웹 테트리스 게임 개발을 완료하였습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임의 구조 및 캔버스 설정
- `style.css`: 게임 UI 및 레이아웃 스타일링
- `src/tetromino.js`: 테트리미노 블록 데이터 및 회전 로직
- `src/board.js`: 게임 보드 관리, 충돌 감지 및 라인 클리어 로직
- `src/renderer.js`: HTML5 Canvas를 활용한 화면 그리기 로직
- `src/main.js`: 게임 루프, 키보드 입력 처리 및 게임 상태 관리

### 구현한 기능 설명
- **클래식 테트리스 로직**: 7가지 테트리미노 생성, 회전, 하강 기능 구현.
- **충돌 감지 및 보드 관리**: 벽, 바닥, 다른 블록과의 충돌을 감지하여 게임 로직 적용.
- **게임 루프**: `requestAnimationFrame`을 사용하여 일정한 속도로 게임 진행.
- **점수 시스템**: 라인 클리어 시 점수가 누적되며, 화면에 실시간 표시.
- **조작**: 키보드 방향키를 사용하여 블록 이동 및 회전 가능.
- **게임 오버**: 블록이 쌓여 더 이상 생성할 수 없을 때 게임 초기화.

### 테스트 결과
- 브라우저를 통해 `index.html`을 열었을 때 정상적으로 게임 화면이 출력됩니다.
- 방향키 조작을 통해 블록이 정상적으로 움직이고 회전합니다.
- 줄이 꽉 찼을 때 라인이 제거되고 점수가 올라가는 것을 확인했습니다.
- 블록이 상단까지 쌓였을 때 게임 오버 알림창이 나타나고 게임이 초기화됩니다.
