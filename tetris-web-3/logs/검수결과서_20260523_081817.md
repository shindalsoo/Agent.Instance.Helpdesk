웹 기반 테트리스 게임 개발을 완료했습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임의 기본 HTML 구조 및 스크립트 연결.
- `src/style.css`: 게임 컨테이너 및 캔버스 스타일링.
- `src/tetrominoes.js`: 테트리미노 모양 및 색상 정의.
- `src/collision.js`: 충돌 감지 로직.
- `src/game.js`: 게임 루프, 블록 이동, 중력 및 렌더링 로직.
- `src/ui.js`: UI 업데이트 로직(향후 확장 가능).

### 구현한 기능 설명
- **게임 엔진**: `requestAnimationFrame`을 사용하여 10x20 그리드 내에서 블록이 자동으로 하강하는 게임 루프를 구현했습니다.
- **조작**: 방향키(좌/우/아래)를 사용하여 블록을 이동할 수 있습니다.
- **충돌 감지**: 벽과 바닥, 그리고 이미 안착된 블록과의 충돌을 감지하여 블록이 겹치지 않도록 처리했습니다.
- **안착 로직**: 블록이 바닥이나 다른 블록에 닿으면 `arena` 배열에 저장되어 게임 보드에 고정됩니다.

### 테스트 결과
- 브라우저 환경에서 `index.html`을 열면 블록이 정상적으로 생성되고 하강합니다.
- 좌/우 이동 키 입력이 정상적으로 동작하며, 벽을 넘지 못하도록 충돌 처리가 되어 있습니다.
- 블록이 바닥에 도달하면 멈추고 다음 블록이 생성되는 기초 흐름이 확인되었습니다.

추가적인 기능(회전, 라인 삭제, 점수 시스템 등)은 현재 구조를 바탕으로 쉽게 확장 가능합니다.
