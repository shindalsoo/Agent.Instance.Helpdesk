# 작업계획서: Web-based Tetris Game 개발

## 1. 프로젝트 개요
본 프로젝트는 웹 브라우저에서 별도의 설치 없이 실행 가능한 클래식한 **Tetris(테트리스) 게임**을 개발하는 것을 목표로 합니다. HTML5, CSS3, 그리고 Vanilla JavaScript를 사용하여 성능이 뛰어나고 구조가 명확한 게임 엔진을 구축합니다.

## 2. 기술 스택
- **Language**: JavaScript (ES6+)
- **Frontend**: HTML5 (Canvas API 또는 DOM Manipulation), CSS3
- **Development Environment**: Browser (Chrome/Edge/Firefox)

## 3. 파일 구조
모든 소스 코드는 `src/` 디렉토리 아래에 관리됩니다.

```text
project-root/
│
├── index.html              # 메인 엔트리 포인트 (HTML 구조)
├── output/
│   └── 작업계획서.md        # 현재 문서
└── src/
    ├── style.css           # 게임 레이아웃 및 디자인 스타일
    ├── game.js             # 핵심 게임 루프 및 엔진 로직
    ├── tetrominoes.js      # 테트리미노(블록) 모양 및 색상 정의
    ├── collision.js        # 충돌 감지 및 회전 알고리즘
    └── ui.js               # 점수, 다음 블록 표시 등 UI 업데이트 로컬
```

## 4. 기능 명세

### 4.1 핵심 게임 엔진 (Core Engine)
- **Grid System**: 10x20 크기의 게임 보드 구현.
- **Tetromino Management**: 7가지 표준 테트리미노(I, J, L, O, S, T, Z) 생성 및 관리.
- **Game Loop**: `requestAnimationFrame`을 사용하여 부드러운 프레임 드랍 없는 애니메이션 구현 및 일정 시간 간격의 자동 하강 로직.
- **Movement**: 좌/우 이동, 소프트 드롭(빠른 하강), 하드 드롭(즉시 바닥 안착) 기능.

### 4.2 물리 및 규칙 (Physics & Rules)
- **Collision Detection**: 벽면 충돌, 바닥 충돌, 이미 쌓인 블록과의 충돌 감지.
- **Rotation Logic**: 블록 회전 시 벽에 걸리는 경우(Wall Kick) 처리.
- **Line Clearing**: 한 줄이 가득 찼을 때 해당 라인을 삭제하고 상단 블록들을 아래로 이동.
- **Scoring System**: 제거된 라인 수에 따른 점수 계산 (1줄 < 2줄 < 3줄 < 4줄 가중치 적용).

### 4.3 UI 및 사용자 경험 (UI/UX)
- **Scoreboard**: 현재 점수, 레벨, 제거된 라인 수 표시.
- **Next Piece Preview**: 다음에 등장할 블록을 미리 보여주는 프리뷰 창.
- **Game Over State**: 블록이 상단 끝에 도달했을 때 게임 종료 처리 및 재시작 버튼 제공.
- **Controls**: 키보드 방향키(Arrow Keys) 또는 WASD를 이용한 조작.

## 5. 구현 우선순위
1.  **P0 (Critical)**: HTML/CSS 기본 레이아웃, 테트리미노 생성, 기본 이동 및 중력 로직, 충돌 감지.
2.  **P1 (High)**: 라인 삭제 기능, 점수 시스템, 게임 종료(Game Over) 처리.
3.  **P2 (Medium)**: 다음 블록 프리뷰, 레벨업 시스템, 회전 보정(Wall Kick).
4.  **P3 (Low)**: 애니메이션 효과, 사운드 이펙트(필요 시), 모바일 터치 컨트롤 추가.

## 6. 테스트 및 QA 계획
- **Unit Test**: 각 테트리미노의 회전 좌표 계산 정확성 검증.
- **Integration Test**: 블록 하강 -> 충돌 -> 라인 삭제로 이어지는 전체 흐름 테스트.
- **Edge Case Test**: 벽면 끝에서의 회전, 하드 드롭 시 즉각적인 충돌 처리, 연속 라인 클리어(Tetris) 상황 테스트.
