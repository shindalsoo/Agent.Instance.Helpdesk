틱택토(Tic-Tac-Toe) 게임 개발을 완료했습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 메인 구조 및 레이아웃 정의
- `src/style.css`: 3x3 그리드 및 게임 UI 스타일링
- `src/script.js`: 게임 상태 관리, 턴 전환, 승리/무승부 판정 및 리셋 로직 구현

### 구현한 기능 설명
- **3x3 격자판**: HTML/CSS를 사용하여 직관적인 게임 보드 구현
- **게임 로직**: 
    - 턴제 게임 (X -> O)
    - 8가지 승리 조건 판정
    - 무승부 판정
    - 이미 클릭한 셀 중복 입력 방지
- **UI/UX**:
    - 현재 차례 및 게임 결과 실시간 표시
    - '다시 시작' 버튼을 통한 게임 초기화 기능

### 테스트 결과
- 브라우저에서 `index.html`을 열어 정상적으로 게임 플레이가 가능함을 확인하였습니다.
- 클릭 시 X/O가 올바르게 표시되며, 승리 조건 충족 시 승자 알림이 정상적으로 출력됩니다.
- 무승부 상황 및 리셋 기능 모두 의도한 대로 작동합니다.
