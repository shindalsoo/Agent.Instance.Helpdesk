테트리스 게임의 기본 구조와 핵심 렌더링 로직을 구현하였습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 화면 레이아웃 및 캔버스 요소 정의
- `src/style.css`: 게임 화면 스타일링
- `src/script.js`: 캔버스 초기화 및 블록 렌더링 로직

### 구현한 기능
- **게임 보드**: 12x20 그리드 시스템 기반의 캔버스 설정
- **렌더링**: Canvas API를 사용하여 블록(테트리미노)을 화면에 출력
- **게임 루프**: `requestAnimationFrame`을 이용한 기본 게임 루프 구조

### 향후 계획
- 블록 이동, 회전, 충돌 감지 로직 구현
- 라인 삭제 및 점수 계산 시스템 추가
- 게임 시작/재시작 기능 완성

현재 기본 렌더링 단계까지 완료되었습니다. 추가 기능 구현을 진행하겠습니다.
