# 프로젝트 개요
본 프로젝트는 HTML5, CSS3, Vanilla JavaScript를 사용하여 웹 브라우저에서 즉시 실행 가능한 클래식 테트리스(Tetris) 게임을 개발하는 것을 목표로 합니다. 별도의 외부 라이브러리 없이 순수 자바스크립트의 Canvas API를 활용하여 가볍고 빠른 게임 환경을 제공합니다.

# 기술 스택
- **Language**: HTML5, CSS3, JavaScript (ES6+)
- **Rendering**: HTML5 Canvas API
- **Environment**: Web Browser (Chrome, Edge, Safari 등)

# 파일 구조
모든 소스 코드는 `src/` 디렉토리 아래에 관리됩니다.

```text
project/
├── index.html          # 게임 실행 메인 페이지
└── src/
    ├── style.css       # 게임 UI 및 레이아웃 스타일링
    └── script.js      # 테트리스 핵심 로직 (게임 엔진)
```

# 기능 명세

### 1. 게임 보드 및 환경 설정
- **Grid System**: 10x20 크기의 그리드 시스템 구축.
- **Canvas Rendering**: Canvas API를 사용하여 블록(Tetromino)과 배경을 화면에 렌더링.
- **Responsive Design**: 브라우저 크기에 따라 적절히 배치되는 레이아웃.

### 2. 테트리스 핵심 로직 (Tetromino)
- **Shape Definition**: 7가지 표준 테트리미노(I, J, L, O, S, T, Z) 및 각 블록의 색상 정의.
- **Movement**: 좌/우 이동, 아래로 빠른 낙하(Soft Drop).
- **Rotation**: 블록의 회전 로직 구현 (벽 충돌 시 회전 제한 처리 포함).
- **Gravity**: 일정 시간 간격으로 블록이 자동으로 하강하는 중력 시스템.

### 3. 게임 규칙 및 상호작용
- **Collision Detection**: 벽면, 바닥, 그리고 이미 쌓인 블록과의 충돌 감지.
- **Line Clearing**: 한 줄이 가득 차면 해당 라인을 삭제하고 점수 부여.
- **Score System**: 라인 제거 시 점수 누적 (한 번에 여러 줄을 지울수록 높은 점수).
- **Game Over**: 블록이 화면 상단 끝까지 쌓여 더 이상 생성될 공간이 없을 경우 게임 종료 처리.

### 4. 사용자 인터페이스 (UI)
- **Score Board**: 현재 점수를 실시간으로 표시.
- **Start/Reset Button**: 게임 시작 및 재시작 기능.
- **Next Piece Preview** (Optional): 다음에 나올 블록을 미리 보여주는 기능.
