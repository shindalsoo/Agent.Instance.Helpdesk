테트리스 게임 개발이 완료되었습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임의 기본 구조 및 UI
- `src/style.css`: 게임 화면 스타일링
- `src/script.js`: 테트리스 게임 엔진 로직 (이동, 회전, 충돌, 점수, 라인 제거 등)

### 구현한 기능
- **게임 엔진**: 12x20 그리드 기반의 게임 환경 구축.
- **테트리미노**: 7가지 표준 블록(I, J, L, O, S, T, Z) 생성 및 랜덤 배치.
- **상호작용**: 키보드 입력(좌/우 이동, 회전, 하강) 처리.
- **게임 규칙**: 블록 충돌 처리, 라인 삭제(가득 찬 줄 제거), 점수 계산, 게임 오버 시 초기화.
- **UI**: 시작 버튼 및 실시간 점수 표시.

### 테스트 결과
- 브라우저를 통해 `index.html`을 열면 게임이 정상적으로 로드됩니다.
- '시작' 버튼을 누르면 블록이 낙하하기 시작합니다.
- 좌/우 이동(좌/우 화살표), 회전(Q/W 키), 빠른 낙하(아래 화살표)가 원활하게 작동합니다.
- 라인을 완성하면 해당 라인이 삭제되고 점수가 증가합니다.
- 블록이 상단까지 쌓이면 게임이 초기화되는 것을 확인했습니다.
