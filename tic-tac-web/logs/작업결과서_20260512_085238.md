**완료된 파일 목록**

| 경로 | 내용 |
|------|------|
| `src/index.html` | 기본 UI(타이틀, 턴 표시, 보드 컨테이너, 재시작 버튼) |
| `src/styles.css` | 그리드 기반 보드 스타일 및 전반적인 레이아웃 |
| `src/app.js` | 게임 로직: 초기화, 클릭 처리, 승자/무승부 판정, 리셋 기능 |

**구현한 기능**

1. **게임판 동적 생성** – 3×3 버튼 그리드  
2. **플레이어 전환 (X/O)** – 번갈아 표시, 이미 선택된 셀 무시  
3. **승리/무승부 판정** – 8가지 패턴 검사 후 `alert`로 결과 표시, 자동 재시작  
4. **재시작 버튼** – 게임 상태와 UI 초기화  
5. **UI 피드백** – 현재 턴 표시, 클릭 시 색상 변화 및 hover 효과

**테스트 결과**

- 브라우저에서 `src/index.html` 열면 보드가 정상적으로 표시됩니다.  
- X/O 입력이 올바르게 반영되고 승리/무승부 판정이 정확히 동작합니다.  
- 재시작 버튼 클릭 시 게임판과 상태가 초기화됩니다.

프로젝트는 이제 완전한 동작형 웹 Tic‑Tac‑Toe 게임입니다!
