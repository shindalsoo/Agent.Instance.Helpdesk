**완료된 파일 목록**

| 경로 | 내용 |
|------|------|
| `src/index.html` | 기본 HTML 구조와 UI(타이틀, 턴 표시, 보드 컨테이너, 재시작 버튼) |
| `src/styles.css` | 그리드 기반 보드 스타일 및 전반적인 레이아웃 |
| `src/app.js` | 게임 로직: 초기화, 클릭 처리, 승자/무승부 판정, 리셋 기능 |

**구현한 기능**

1. **게임판 생성** – 3×3 그리드를 동적으로 버튼으로 만들고 이벤트 연결  
2. **플레이어 전환** – X와 O가 번갈아 표시, 이미 선택된 셀은 무시  
3. **승리/무승부 판정** – 8가지 패턴을 검사해 승자 또는 무승부를 감지하고 알림 표시 후 자동 재시작  
4. **재시작 버튼** – 게임 상태와 UI를 초기화  
5. **UI 피드백** – 현재 플레이어 표시, 클릭 시 색상 변화 및 hover 효과

**테스트 결과**

- 브라우저에서 `src/index.html`을 열면 3×3 보드가 정상적으로 나타남  
- X/O 입력이 올바르게 반영되고 승리/무승부 판정이 정확히 동작함  
- 재시작 버튼 클릭 시 게임판과 상태가 초기화됨  

> **주의**: 현재 `alert()`로 결과를 표시하고 자동으로 리셋하도록 구현했습니다. 필요에 따라 모달이나 별도 메시지 영역을 추가할 수 있습니다.
