**생성/수정한 파일 목록**

- `index.html`  
- `src/style.css`  
- `src/utils.js`  
- `src/board.js`  
- `src/app.js`

---

**구현한 기능 설명**

1. **기본 HTML 구조** (`index.html`)  
   - 게임 타이틀, 점수 표시, 재시작 버튼, 격자 컨테이너 및 Game Over 메시지를 포함합니다.

2. **스타일링** (`style.css`)  
   - Flexbox와 CSS Grid를 이용해 4x4 격자를 만들고, 기본 블록 스타일(배경색·폰트)을 정의했습니다.

3. **유틸리티 함수** (`utils.js`)  
   - 빈 셀 찾기(`getEmptyCells`)와 랜덤 타일 추가(`addRandomTile`)를 제공합니다.

4. **격자 관리 및 렌더링** (`board.js`)  
   - 2D 배열 형태의 격자를 초기화하고 두 개의 랜덤 블록을 생성합니다.  
   - `render()` 메서드로 DOM에 격자와 블록 값을 표시합니다.

5. **게임 로직 제어** (`app.js`)  
   - 게임 시작(`startGame()`)과 재시작 버튼 이벤트를 연결했습니다.  
   - 현재는 키 입력 및 이동/병합 로직은 구현되지 않았으나, 추후 `board.js`에 방향 처리 메서드를 추가하면 쉽게 확장됩니다.

---

**테스트 결과**

1. 브라우저에서 `index.html`을 열면 4x4 격자가 보이고 초기 두 칸에 2 또는 4가 표시됩니다.  
2. **Restart** 버튼 클릭 시 게임이 새로 시작됩니다.  

> 현재는 이동/병합 로직이 없으므로 실제 플레이는 불가능하지만, 추가 기능을 쉽게 구현할 수 있는 기반 구조를 제공했습니다.
