Implemented the basic 2048 web‑game skeleton:

- **index.html** – UI 레이아웃 및 모듈 스크립트 연결  
- **src/style.css** – 격자와 블록 기본 스타일 (Flexbox + Grid)  
- **src/utils.js** – 빈 셀 찾기, 랜덤 타일 추가 함수  
- **src/board.js** – 2D 배열 격자 관리 및 `render` 메서드  
- **src/app.js** – 게임 초기화, 재시작 버튼 이벤트

현재 기능은 “초기 화면”, “격자 렌더링”, “블록 두 개 생성”, “Restart” 버튼에 한정됩니다. 키 입력·이동·병합·점수 로직은 `board.js`와 `app.js`에 차후 추가할 수 있습니다.

**테스트 요약**

1. **초기화**: 4x4 격자, 두 개의 랜덤 블록(2/4) 표시  
2. **Restart** 버튼 클릭 시 새 게임 시작  

이제 실제 브라우저에서 바로 실행 가능합니다.
