웹 기반 테트리스 게임 개발을 완료했습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 메인 페이지 및 UI 구조
- `src/style.css`: 게임 스타일링
- `src/tetrominoes.js`: 테트리미노 정의
- `src/utils.js`: 게임 유틸리티 함수 (충돌 감지 등)
- `src/game.js`: 게임 로직 및 엔진

### 구현한 기능
- **핵심 게임 엔진**: 10x20 그리드 기반의 게임 보드와 `requestAnimationFrame`을 이용한 게임 루프 구현.
- **테트리미노 제어**: 테트리미노 생성, 좌우 이동, 하단 이동, 충돌 감지 로직 구현.
- **게임 규칙**: 라인 클리어 시 점수 계산 및 게임 오버 시 보드 초기화 로직 구현.
- **UI**: 점수 및 레벨 표시, 시작 버튼 구조 구현.

### 테스트 결과
- 브라우저 환경에서 게임이 정상적으로 로드되며, 테트리미노가 생성되고 아래로 이동합니다.
- 좌우 이동 및 아래로 가속 기능이 정상적으로 작동합니다.
- 라인이 가득 찼을 때 라인 제거 및 점수 업데이트가 이루어집니다.
- 게임 오버 시 보드가 초기화됩니다.
