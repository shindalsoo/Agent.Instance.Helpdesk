# 개발 작업계획서: 웹 기반 테트리스(Tetris) 게임 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 HTML5, CSS3, Vanilla JavaScript를 사용하여 브라우저에서 별도의 설치 없이 실행 가능한 클래식 테트리스 게임을 개발하는 것을 목표로 합니다. 사용자는 키보드를 통해 블록(Tetromino)을 조작하여 한 줄을 가득 채워 제거하고, 높은 점수를 획득해야 합니다.

## 2. 기술 스택
- **Frontend**: HTML5 (Canvas API), CSS3
- **Logic**: JavaScript (ES6+)
- **Development Environment**: Browser (Chrome/Edge/Safari)

## 3. 파일 구조
모든 소스 코드는 `src/` 디렉토리 아래에 위치합니다.

```text
project-root/
│
├── index.html              # 메인 엔트리 포인트 및 UI 레이아웃
├── src/
│   ├── style.css           # 게임 스타일링 및 애니메이션
│   ├── script.js          # 핵심 게임 로직 (Game Loop, Collision, Scoring)
│   ├── constants.js        # 게임 설정값 (Grid size, Colors, Tetromino shapes)
│   └── utils.js            # 보조 유틸리티 함수 (Random generation 등)
└── README.md               # 프로젝트 설명 및 실행 방법
```

## 4. 기능 명세

### 4.1. 게임 보드 및 그래픽 (Canvas API)
- **Grid System**: 10x20 사이즈의 그리드 시스템 구축.
- **Rendering**: `requestAnimationFrame`을 이용한 부드러운 프레로 렌더링 루프 구현.
- **Visuals**: 각 테트로미노 블록에 고유 색상 적용 및 격자 라인 시각화.

### 4.2. 테트로미노(Tetromino) 로직
- **Shapes**: I, J, L, O, S, T, Z 7가지 표준 모양 구현.
- **Rotation**: 블록의 회전 알고리즘 구현 (Wall Kick 최소 기능 포함).
- **Movement**: 좌/우 이동, 아래로 빠른 낙하(Soft Drop), 즉시 낙하(Hard Drop) 기능.

### 4.3. 게임 메카닉 (Core Mechanics)
- **Collision Detection**: 벽면 충돌 및 쌓여있는 블록과의 충돌 감지.
- **Line Clearing**: 가로 한 줄이 모두 채워질 경우 해당 라인 삭제 및 상단 블록 하강.
- **Scoring System**: 제거한 라인 수에 따른 점수 계산 (1줄 < 2줄 < 3줄 < 4줄 가중치 적용).
- **Leveling**: 특정 점수 도달 시 레벨업 및 블록 낙하 속도 증가.
- **Game Over**: 새로운 블록이 생성될 공간이 없을 경우 게임 종료 처리.

### 4.4. 사용자 인터페이스 (UI/UX)
- **Scoreboard**: 현재 점수, 레벨, 제거된 라인 수를 실시간 표시.
- **Next Piece Preview**: 다음에 나타날 블록을 미리 보여주는 기능.
- **Control Instructions**: 조작법(화살표 키, Space 등) 안내.
- **Start/Restart Button**: 게임 시작 및 재시작 버튼.

## 5. 구현 우선순위

1.  **P0 (Must Have)**:
    *   HTML Canvas 기반의 기본 그리드 렌더링.
    *   테트로미노 생성 및 기본 이동(좌, 우, 하).
    *   충돌 감지 로직(벽, 바닥).
    *   라인 제거 기능 및 점수 시스템.
2.  **P1 (Should Have)**:
    *   블록 회전 기능.
    *   레벨업에 따른 속도 증가 메카닉.
    *   Game Over 상태 처리.
3.  **P2 (Nice to Have)**:
    *   Next Piece 미리보기 UI.
    *   Hard Drop 기능 (Space 키).
    *   사운드 효과 및 애니메이션 효과.
