테트리스 게임 개발을 시작했습니다. 현재 HTML, CSS, 기본적인 렌더링 로직(Canvas API)이 구현되어 있습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 기본 레이아웃 구성
- `src/style.css`: 게임 화면 스타일링
- `src/constants.js`: 게임 설정값(그리드, 색상, 블록 모양) 정의
- `src/utils.js`: 유틸리티 함수(행렬 생성)
- `src/script.js`: 게임 렌더링 루프 및 초기 상태 설정

### 구현한 기능
- 10x20 그리드 시스템 기반의 렌더링
- 테트로미노(Tetromino) 블록 데이터 구조화
- `requestAnimationFrame`을 이용한 기본 게임 루프

### 다음 단계
- 테트로미노의 이동, 회전, 충돌 감지 로직 구현
- 라인 제거 및 점수 시스템 구현

테스트를 위해 브라우저에서 `index.html`을 열어 캔버스에 기본 블록이 렌더링되는지 확인할 수 있습니다.
