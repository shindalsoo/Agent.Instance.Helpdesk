이 프로젝트의 1단계 및 2단계(기초 환경 구축 및 기본 게임 엔진 구조)를 성공적으로 완료했습니다.

### 생성/수정한 파일 목록
- `index.html`: 게임 메인 페이지 구조 정의
- `src/style.css`: 게임 UI 스타일링
- `src/constants.js`: 게임 상수(블록 모양, 색상) 정의
- `src/utils.js`: 보조 함수 정의
- `src/game.js`: 게임 루프 및 캔버스 렌더링 로직

### 구현한 기능 설명
- HTML5 Canvas를 활용한 게임 화면 렌더링 환경 구축
- 테트리스 그리드(10x20) 및 7가지 테트리미노 데이터 구조화
- `requestAnimationFrame`을 사용한 기본 게임 루프 구현

### 향후 계획
다음 단계로 블록의 이동, 회전 로직, 충돌 감지 및 라인 클리어 기능을 구현할 예정입니다. 현재까지의 구조는 기본적인 렌더링 프레임워크를 갖추었으며, 이제 실제 게임 플레이 로직을 삽입할 준비가 되었습니다.
