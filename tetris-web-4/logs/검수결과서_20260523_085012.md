이 프로젝트의 3단계(게임 규칙 적용)까지 성공적으로 완료했습니다.

### 생성/수정한 파일 목록
- `src/utils.js`: 충돌 감지(`collide`), 블록 합치기(`merge`), 회전(`rotate`) 로직 구현
- `src/game.js`: 플레이어 조작(이동, 회전, 드롭), 라인 클리어(`arenaSweep`), 점수 계산, 게임 루프 통합

### 구현한 기능 설명
- **게임 엔진 고도화**:
    - `collide`: 벽, 바닥, 기존 블록과의 충돌을 정확히 감지하여 이동/회전 제한
    - `merge`: 플레이어의 블록이 바닥에 닿으면 고정하여 `arena` 배열에 병합
    - `rotate`: 행렬 연산을 통한 블록 회전 및 벽 끼임 방지 로직(Wall Kick) 적용
    - `arenaSweep`: 한 줄이 꽉 찼을 때 삭제하고 점수를 갱신하는 라인 클리어 로직
- **플레이어 조작**:
    - 좌/우 이동 (화살표 키)
    - 회전 (Q/W 키)
    - 하드/소프트 드롭 (아래 화살표)
