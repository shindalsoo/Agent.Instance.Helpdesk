### 생성/수정한 파일 목록
- `index.html`: 게임 메인 페이지 UI 구성 (점수판, 시작 버튼, 게임 오버 표시)
- `src/style.css`: 게임 레이아웃 및 UI 스타일링 고도화
- `src/constants.js`: 테트리미노 상수 정의
- `src/utils.js`: 게임 로직 보조 함수 (행렬, 충돌, 회전)
- `src/game.js`: 게임 루프, 플레이어 조작, 상태 관리, UI 연동 로직

### 구현한 기능 설명
- **게임 엔진**: 10x20 그리드 시스템, 7가지 테트리미노, 행렬 기반 회전 및 중력 시스템.
- **게임 규칙**: 실시간 충돌 감지, 라인 클리어 및 점수 계산, 게임 오버 조건 처리.
- **UI/UX**: 
    - 게임 시작/일시정지/재시작 기능 구현.
    - 게임 오버 시 알림 및 재시작 UI 활성화.
    - 점수판 실시간 업데이트.
- **조작**: 방향키(이동, 드롭), Q/W(회전) 키보드 이벤트 구현.
