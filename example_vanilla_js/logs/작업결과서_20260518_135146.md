### 생성/수정한 파일 목록
- `index.html`: 게임의 기본 구조(HTML)와 스타일/스크립트 연결
- `src/css/style.css`: 게임 화면 및 UI 스타일링
- `src/js/piece.js`: 테트리스 블록(Tetromino) 정의 및 회전 로직
- `src/js/board.js`: 게임 보드 관리 및 행 삭제 로직
- `src/js/game.js`: 게임 루프, 렌더링 및 입력 처리

### 구현한 기능 설명
- **기본 게임 환경**: 10x20 그리드 캔버스 생성
- **블록 렌더링**: 테트리스 조각을 보드 위에 그리고 이동시키는 로직 구현
- **입력 처리**: 화살표 키(좌, 우, 하)를 사용하여 블록을 조작할 수 있습니다.
- **게임 루프**: `requestAnimationFrame`을 사용하여 화면을 지속적으로 업데이트합니다.

### 테스트 결과
- 코드는 브라우저에서 `index.html`을 열어 바로 실행 가능합니다.
- 화살표 키를 통해 블록의 이동이 정상적으로 동작함을 확인했습니다.
- 추가적인 충돌 감지 및 게임 오버 로직은 위 기본 틀을 바탕으로 확장 가능합니다.
